Macro-mecânica - Estudo de várias lâminas (laminado)

Considerações:

  • Uma laminado é composto por uma ou mais lâminas.
  • Na macro-mecânica existe o interesse calcular certas propriedades como flexibilidade, rigidez, tensões, deformações, tanto globais como locais.
  • Esta área possibilita o cálculo de propriedades do compósito decorrentes do empilhamento de lâminas.
  • A lógica dos cálculos considerou que as lâminas teriam os mesmos materiais, alterando apenas o ângulo entre elas.
  • Para compor o laminado, insira no campo de ângulos, cada lâmina separada por uma barra "/". Exemplo: 0/30/90/90/30/0.
  • Para o uso da ferramenta abaixo insira os valores nas caixas.
  • Para números decimais use ponto ao invés de vírgula.
  • Para alterar algum valor bloqueado, clique duas vezes para torná-lo editável novamente.
  • Os cálculos ocorrem toda vez que um valor é adicionado ou alterado.
